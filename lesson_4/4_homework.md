# Домашнее задание 4: SCSS + Vite проект

## Цель

Освоить современную разработку с использованием SCSS препроцессора и сборщика Vite. Настроить сборку и научиться использовать продвинутые возможности SCSS для создания масштабируемой системы стилей.

## Что нужно сделать

### Создание Vite проекта

Создайте новый проект с помощью Vite, используя vanilla JavaScript шаблон. 

### Миксины и функции

Разработайте набор полезных миксинов для типичных задач: создание кнопок разных стилей, медиа-запросы, флекс-центрирование, анимации переходов. Создайте функции для работы с цветами, размерами и расчетами. Миксины должны быть параметризованы и покрывать основные паттерны использования в проекте.

## Дополнительные задания

### Продвинутые SCSS возможности

Изучите и примените продвинутые возможности SCSS: циклы (@for, @each, @while), условия (@if, @else), интерполяцию переменных, плейсхолдеры (@extend). Создайте генераторы утилитарных классов для отступов, цветов, размеров шрифтов.

## Критерии выполнения

Проект должен корректно собираться и запускаться через Vite команды, иметь правильную архитектуру SCSS файлов с логическим разделением ответственности, использовать продвинутые возможности SCSS для создания масштабируемой системы стилей. Все компоненты должны быть стилизованы согласованно с использованием общих переменных и миксинов.
